# KukhtamIE-72informatic_project
Этот проект для Итоговой контрольной в 28.05.2026
Проект называется "Газоанализатор воздуха на открытых пространствах"
Этот проект направлен на анализ загрязнения воздуха в разных частях города. После сбора данных они направляются в базу данных где сохраняются и создаются графики основанные на показаниях собранных с датчиков.
